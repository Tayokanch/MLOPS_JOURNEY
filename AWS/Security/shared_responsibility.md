# AWS Share Responsibility Model

## CUSTOMERS RESPOMSIBILITIES
- Platform
- Applications
- Identity and Access Management
- Operating System, Network and Firewall Configuration
- Client-side Data Encryption and Data Integrity Authentication
- Server-Side Encryption (File System / Data)
- Networking Traffic Protection (Encrption, Integrity, Identity)


## AWS RESPONSIBILITIES
`Software Resposibitiliess`
- Storage
- Database
- Networking
  
`Hardware / AWS Global Infrastrucure Responsibilities`
- Regions
- Availability Zones
- Edge Locations
