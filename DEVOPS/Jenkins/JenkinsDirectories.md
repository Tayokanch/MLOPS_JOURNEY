- `/var/lib/jenkins/workspace/` : This is where all the jenkins `Jobs Directory` are created
  
- `ll` : This is used to list all the `jobs` inside the `/workspace/`
  
- `/var/lib/jenkins/workspace/job_directory/target` : This is where the build file is saved after `building pipeline` has completed e.g nodejs would have `/dist`, Java would have a `.jar or war file`

-  `/var/lib/jenkins/workspace/job_directory/target/surefire-reports`: This is the Dir where all `pipeline reports` are saved.
