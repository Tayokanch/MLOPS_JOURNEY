# PLUGINS

- These are extentions that superchagrge Jenkins with functionalities for various tools and technologies you use in your development process.

- plugins are saved in the `/var/lib/jenkins/plugins`

# HOW TO ADD OR REMOVE PLUGINS
- Click on the `settings Icon` to go to `manage jenkins` on the dahsboard, then select `plugins`


# NAMES OF SOME PLUGINGS
- `Copy Artifact`: This plugging allows us to be able to copy any artifact / file which we had generate from on build/stage to another stage

- `Yet Another Build Visualizer`: This Visualises the flow of downstream build on the projects and build pages
